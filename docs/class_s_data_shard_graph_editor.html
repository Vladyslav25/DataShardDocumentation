Detailed Description

Schema for the DataShard graph Defines how nodes can be connected and interacted with Graph representation for DataShard relationships Contains nodes representing Actors and DataShards with their connections Node representing an Actor/Owner in the DataShard graph Has output pins for each DataShard key Node representing a DataShard object Has an input pin that can connect to Actor outputs Widget for editing and visualizing the DataShard graph Allows viewing relationships between Actors and DataShards
