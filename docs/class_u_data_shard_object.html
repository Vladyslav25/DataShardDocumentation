Detailed Description

DataShardObject is the core container class for data in the DataShard system

This class provides a flexible data storage mechanism that can hold any Unreal property type. It uses a type-erased byte buffer approach with a property reference to correctly interpret the stored data. The system maintains these objects in a pool for efficient reuse.

Key capabilities:

  • Store and retrieve any UProperty data type
  • Maintain type safety through property system
  • Support change notifications
  • Efficiently manage memory through the pool system
